# Yunoga

A fixed-width font similar to Courier New, but a little thicker, less crispy and more smooth.

The Yunoga font is a download of the *Courier 10 Pitch* font. The *Courier 10 Pitch* font is a
property of Bitstream Inc.

Converted to TTF and WOFF2 from
http://xorg.freedesktop.org/releases/individual/font/

## The font name

Note that installing the font locally in your system will install it by the "Courier 10 Pitch" name.
